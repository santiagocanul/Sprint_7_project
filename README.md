Sprint 7 Project

Descripción

En este proyecto se creó una aplicación web sencilla para hacer un análisis exploratorio de un conjunto de datos de anuncios de autos en Estados Unidos. La aplicación permite visualizar la información de forma interactiva usando gráficos.

El objetivo principal es explorar los precios de los vehículos y observar la relación entre algunas variables como el precio y el kilometraje.

Datos

Se utilizó un archivo CSV que contiene información sobre anuncios de vehículos, como:
	•	Precio
	•	Año del vehículo
	•	Kilometraje
	•	Fabricante
	•	Tipo de vehículo
	•	Región

Los datos se cargan y procesan usando la librería pandas.

Aplicación web

La aplicación fue desarrollada con Streamlit y Plotly. Incluye:
	•	Un encabezado con el título del proyecto
	•	Un histograma para ver la distribución de precios
	•	Un gráfico de dispersión para analizar la relación entre el precio y el odómetro
	•	Botones (o casillas) para mostrar los gráficos de forma interactiva

Cómo ejecutar la aplicación
	1.	Clonar el repositorio.
	2.	Activar el entorno virtual.
	3.	Instalar las dependencias:
        - pip install -r requirements.txt
    4.	Ejecutar la aplicación:
        - streamlit run app.py
    
Después de esto, la aplicación se abre en el navegador.
